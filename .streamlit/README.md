# Streamlit secrets

Para testar localmente com mais segurança:

1. Copie o template:
   ```bash
   cp .streamlit/secrets.toml.example .streamlit/secrets.toml
   ```
2. Preencha os valores reais no arquivo `.streamlit/secrets.toml`.
3. Rode a aplicação.

> O arquivo `.streamlit/secrets.toml` está no `.gitignore` e não será versionado.
